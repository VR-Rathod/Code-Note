## 1. History
-
	- **How**: Adobe XD was developed by Adobe Inc., first released in 2016 as part of the Adobe Creative Cloud suite.
	- **Who**: The software was created by a team at Adobe focused on UI/UX design tools.
	- **Why**: Adobe XD was designed to provide an all-in-one solution for designing and prototyping user experiences, enabling designers to create interactive prototypes and collaborate effectively.
-
- ## 2. Intro
-
	- Adobe XD is a powerful design and prototyping tool used for creating user interfaces and experiences for web and mobile applications. It offers features such as vector design, interactive prototypes, and collaboration tools. Adobe XD integrates well with other Adobe products, making it a popular choice for designers already in the Adobe ecosystem.
-
	- ### Advantages
		- Intuitive user interface tailored for UI/UX design.
		- Seamless integration with other Adobe Creative Cloud applications.
		- Interactive prototyping capabilities with transitions and animations.
		- Collaboration features that allow for real-time feedback and sharing.
	-
	- ### Disadvantages
		- Limited offline capabilities compared to some desktop applications.
		- Some features may require an Adobe Creative Cloud subscription.
		- Performance can vary with very large or complex projects.
-
- ## 3. Notes
-
	- **Basic Navigation**: Use the spacebar to pan around the canvas; Cmd + D (Mac) or Ctrl + D (Windows) to duplicate elements; Cmd + E (Mac) or Ctrl + E (Windows) to export assets.
	- **Design Tools**: Familiarize yourself with the shape tools, text tools, and repeat grid feature for efficient design.
	- **Shortcuts**: Learn essential shortcuts (e.g., Cmd + Z for undo, Cmd + G for grouping) to enhance workflow efficiency.

-
	- **Vector Design**: Advanced vector drawing tools for precise design work.
	- **Prototyping**: Create interactive prototypes with transitions, gestures, and animations.
	- **Design Systems**: Build and maintain design systems with reusable components and styles.
	- **Collaboration**: Share designs and prototypes for feedback with stakeholders in real time.
	- **User Testing**: Features to facilitate user testing and feedback gathering on prototypes.
	- **Integration**: Works seamlessly with other Adobe tools like Photoshop and Illustrator.
